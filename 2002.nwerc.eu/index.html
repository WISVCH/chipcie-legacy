Intro

This is the official website for the North Western European Programming Contest 2002 in Delft, The Netherlands. This is a programming contest of the ACM. A programming contest is a contest where teams from different universities try to solve a number problems by writing a smart computer program. The team that solves the most problems in the least amount of time wins.

If you are new to this phenomenon, you might want to read some more introductory material in the contest section.
