Computers

The computers used during the contest are all Intel-compatible machines. Each will be equipped with a processor operating at several hundred Megahertz, at least 64 MB RAM and a lots-of-gigabytes hard disk. These computers should be more than adequate to compute the solution to any of the problems if a solution is correct. Solutions that are not efficient enough will not be able to run in the required time limit regardless of the speed of the computer.

Operating Systems

Teams will have a choice between the Debian/GNU Linux OS, from now on referred to as 'Linux', or an OS from Microsoft (Windows NT 4), from now on referred to as 'Windows'.

Jury system

The jury system is a suite of applications that allows submission of solutions and communication with the jury.

Submission
Submission is done using command line tools that send the solution over the network to the jury. Note: ALL submissions will be judged on a linux judging machine.

Communication
A tool called 'TeamTool' is used to send messages from the jury to the teams and vice versa.
